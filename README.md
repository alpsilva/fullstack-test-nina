# fullstack-test-nina
Template repository for fullstack web developers at Super NINA.

# Backend
## Requirements
- Python 3.11+

## Installing dependencies
From the root of the project, run:
```sh
pip install -r back/requirements.txt
```

## Running server
From the root of the project, run:
```sh
fastapi dev back/app.py
```
